GlotPress plugin to permanently delete unwanted translations.

== Description ==

GlotPress plugin to permanently delete unwanted translations.
This plugin adds a button to the translation editor interface that allows users with the appropriate permissions to delete translations permanently from the database.


= Configuration =

Once you have installed GP Delete Translation it is ready to use.
